Einstellungen speichern & laden

Hier können sie die gegenwärtig verwendeten Einstellungen in einer Datei auf Ihrem PC speichern.

Außerdem können Sie zuvor gespeicherte Einstellungen, welche auf Ihrem PC gespeichert sind, wieder laden.

Größe des mittleren Polygons

Stellen Sie hier die Größe des Polygons in der Mitte ein.

Dieser Wert wird durch die Auflagefläche Ihres Spiegels bestimmt.

px

Größe der Bilder

Stellen Sie hier die Größe der Bilder ein.

Es gibt nur eine Größe und der Wert wird für alle Bilder übernommen.

px

Position der Bilder

Mit dieser Einstellung können Sie die Position der Bilder einstellen.

Dafür wird ein Offset-Wert verwendet. Dadurch wird das Bild um den eingestellten Wert von seiner Ausgangsposition aus verschoben.

Es gibt nur ein Offset und der Wert wird für alle Bilder übernommen.

px

Anzahl der Seiten bzw. Bilder

Mit dieser Einstellung können Sie die Anzahl der Seiten einstellen, die das Polygon hat.

An jeder Seite wird ein Bild dargestellt. Die Anzahl sollte zwischen 3 und 20 liegen.

Sie können max. so viele Bilder verwenden, wie das Polygon Seiten hat. Wenn Sie weniger Bilder verwenden, dann wiederholen sich die Bilder.

Seiten

Zeit zwischen den Bildwechseln

Dieser Wert ist die Zeit (in Millisekunden), welcher zwischen den Wechsel vom aktuellen zum nächsten Bild liegt.

Dieser Wert ist nur wichtig, wenn Sie als "Bild" ein GIF hochgeladen haben.

Später folgt die Unterstützung für Videos. Dann kann auch die Zeit zwischen dem Anzeigen der Bilder des Videos hier festgelegt werden.

ms
