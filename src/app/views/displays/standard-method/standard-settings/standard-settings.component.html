Size of the images

Set the scaling factor for the images here.

%

Position of the images

With this setting you can adjust the position of the images.

For this an offset value is used. This moves the image by the set value from its starting position.

There is only one offset and the value is taken for all images.

px

Number of sides / images

With this setting you can adjust the number of sides the polygon has.

Each side displays an image. The number should be between 3 and 10.

You can use as many images as the polygon has sides. If you use fewer images, the images will be repeated.
